## 验证命令

```powershell
$ErrorActionPreference = 'Stop'
npm test
npm run build
git diff --check
```

浏览器联调执行一次 `npm run dev`，会同时启动 API 与 Web。配置 `DATABASE_URL` 时，知识库状态保存在 PostgreSQL 的 `smarthub` schema；未配置时按 `SMARTHUB_DATA_FILE` 回退到 JSON 文件。上传原文件和模型缓存仍写系统默认目录，运行数据不提交 Git。远程 Embedding 来源由当前知识库配置维护；来源地址和 API Key 随配置版本保存，读取和保存响应不回显密钥，任务与错误信息会脱敏。
